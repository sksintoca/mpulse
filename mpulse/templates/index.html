mPulse Mobile Test API service

API Routes

Read Members by:

/member/ Primary Key Id
/member/id/ Primary Key Id
/members/phone/ Phone Number
/members/clientid/ Client Member Id
/members/ Read list of members
/members/account/ Read list of member for account

Create new members

/member methods=["POST"]
/upload methods=['POST']

Examples:
curl -i -X POST -H "Content-Type: multipart/form-data" -F "file=@member_data.csv" http://localhost:5000/upload
curl -d "@data.json" -H "Content-Type: application/json" -X POST http://localhost:5000/member

Delete member records

/members/clear methods=["DELETE"]
/members/clear/ methods=["DELETE"]

Examples:
curl -X DELETE http://localhost:5000/members/clear
